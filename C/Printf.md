# Formatos Específicos 

| Simbolo | Elemento do tipo                    |
| ------- | ----------------------------------- |
| %c      | char                                |
| %d      | int (decimal)                       |
| %f      | float / double                      |
| %i      | int (pode ser octal ou hexadecimal) |
| %u      | unsigned int                        |
| %s      | string                              |
| %x      | número hexadecimal                  |
| %%      | Exibe a porcentagem                 |
| \%      | Exige a porcentagem                 |
| %Lf     | long double                         |
